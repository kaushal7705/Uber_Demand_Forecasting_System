# Uber_Demand_Forecasting_System
Machine Learning based demand forecasting system using Python, FastAPI, Streamlit, feature engineering, and real-time prediction.

Objective :-
       To develop an intelligent demand prediction system for urban mobility and decision support.

Steps of Approach :-
       Collected and explored historical ride demand data

       Performed data cleaning and preprocessing

       Extracted time-based features such as hour, day, and month

       Trained machine learning model for demand prediction

       Evaluated model performance using R² score

       Built FastAPI backend for prediction service

       Developed Streamlit dashboard for user interaction
